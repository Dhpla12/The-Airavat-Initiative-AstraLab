# AstraLab Downloads

The macOS DMG is committed here so GitHub Pages can serve it directly:

- `AstraLab-1.0.0.dmg`
- `AstraLab-1.0.0.dmg.sha256`

Windows and Linux installers are built by `.github/workflows/release-installers.yml`
on GitHub-hosted Windows and Linux runners. Their website links point to the latest
GitHub Release assets:

- `AstraLab-1.0.0.msi`
- `AstraLab-1.0.0.AppImage`
